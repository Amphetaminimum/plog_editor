# Demo photographs

These twelve optimized photographs are bundled solely as the Plog Kyoto demonstration set. They begin after human curation and are used to demonstrate story planning, ordering, editable blocks, layout, and export.

The photographs are not covered by the repository's MIT software license. Copyright remains with the project author; all rights are reserved for the image assets.

The `kyoto-demo-*.jpg` files are web-sized copies with a 2600 px maximum edge. Full-resolution `DSC*.jpg` source files are intentionally ignored by Git and should not be deployed.
